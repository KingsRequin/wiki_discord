Introduction

Bienvenue sur ce wiki dédié au formatage Markdown dans Discord. Vous trouverez ici les principaux styles de texte que vous pouvez utiliser dans vos messages pour les rendre plus lisibles et percutants. Chaque section explique la syntaxe à employer, fournit un exemple de code et montre le rendu attendu.

Texte en gras

Pour souligner un passage important, entourez‑le de deux astérisques. Discord appliquera automatiquement une mise en gras du texte.

**Ceci est un texte en gras**
Ceci est un texte en gras

Italique

L’italique est utile pour mettre en avant des titres d’œuvres ou ajouter une nuance dans votre message. Entourez le texte d’un astérisque pour appliquer ce style.

*Ceci est en italique*
Ceci est en italique

Barré (strikethrough)

Pour rayer une partie de votre message tout en laissant la lecture du texte original, utilisez deux tildes (~) au début et à la fin de la portion concernée.

~~Je change d’avis~~
Je change d’avis

Souligné

Pour souligner un mot ou une expression, placez deux traits bas ( _ ) avant et après le texte.

__Titre souligné__
Titre souligné

Spoiler

Pour masquer temporairement une information (par exemple un spoiler), encadrez‑la avec deux barres verticales (pipes). Le texte ne sera visible qu’après un clic.

||Révélation secrète||
Révélation secrète

Titres (headers)

Pour hiérarchiser un long message, utilisez des dièses (#) en début de ligne. Plus il y a de dièses, plus le titre est petit.

# Titre niveau 1
## Titre niveau 2
### Titre niveau 3
#### Titre niveau 4

Titre niveau 1

Titre niveau 2

Titre niveau 3

Titre niveau 4

Liens cliquables

Pour créer un lien hypertexte sur un mot, entourez le mot de crochets et placez l’URL entre parenthèses juste après. Discord affichera le mot souligné en bleu et le rendra cliquable.

[Visitez Discord](https://discord.com/)
Visitez Discord

Listes

Pour créer une liste à puces, commencez chaque ligne par un tiret (-) ou un astérisque (*). Pour une liste numérotée, utilisez 1., 2., 3., etc. Discord convertira automatiquement votre texte en liste.

- Premier élément
- Deuxième élément
- Troisième élément

1. Étape une
2. Étape deux
3. Étape trois
  • Premier élément
  • Deuxième élément
  • Troisième élément
  1. Étape une
  2. Étape deux
  3. Étape trois

Citation

Pour mettre en avant une citation ou une remarque, commencez la ligne par le signe supérieur (>) suivi d’un espace. Discord affichera le texte sur un fond distinct.

> Ceci est une citation inspirante
Ceci est une citation inspirante

Code en ligne

Pour mettre en valeur un mot ou une expression de code dans une phrase, entourez le texte d’un seul backtick (`). Il sera rendu dans une police monospace.

`console.log("Bonjour")`
console.log("Bonjour")

Bloc de code

Pour partager plusieurs lignes de code, utilisez trois backticks avant et après votre bloc. Vous pouvez spécifier un langage pour bénéficier d’une coloration syntaxique.

```python
def saluer(nom):
    print(f"Bonjour {nom} !")

saluer("Alice")
```
def saluer(nom):
    print(f"Bonjour {nom} !")

saluer("Alice")
